# Contributing to Casio Theme

Thanks for contributing ✨

## Local setup
```bash
pnpm install
pnpm run lint:check
pnpm run format:check
```

## Workflow
1. Create a branch from `main`.
2. Keep changes scoped (palette, UI colors, docs).
3. Update screenshots when visual output changes.
4. Run checks before opening a PR.

## Pull request checklist
- [ ] Theme JSON is valid and linted
- [ ] Formatting checks pass
- [ ] README/changelog updated when needed
- [ ] Screenshots updated when visual changes are introduced
